# Personal Portfolio Website

The hosted website is: https://yuyang-liu-personal-website.vercel.app/

## Summary

- **Initial Design** Built with Vercel, V0 using a structured design prompt.
- **Interactive Components** - Includes sections for intro, projects and feedback
- **Database Integration** - Supabase for backend services.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) 16.1.6
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [PostCSS](https://postcss.org/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) with [shadcn/ui](https://ui.shadcn.com/)
- **Database**: [Supabase](https://supabase.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: React Hook Form
- **Language**: TypeScript
